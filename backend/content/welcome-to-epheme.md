Epheme is a living collection of ideas, code, and patterns for developers who want software that is lighter, more private, and easier to own.

We share short posts, tutorials, and examples here so you can use the same themes in your own projects. Expect practical notes, open source experiments, and the occasional design sketch.

This blog is built from markdown in the repo and rendered by a simple Express-powered site, so the content is easy to edit and version.
